# My Agent Skills

自定义 Agent Skills 集合，适用于 Codex / Claude Code 等 AI Agent。

## 安装

```bash
npx skills add berial/skills --skill '*' --agent universal --yes
```

## Skills

| Skill | Description |
|-------|-------------|
| [dart-generate-doc-comments](skills/dart-generate-doc-comments/) | 为Dart代码生成Dartdoc API文档注释，支持中英文 |
